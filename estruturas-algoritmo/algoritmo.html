O que é um Algoritmo?

Definição de Algoritmo

Um algoritmo é uma sequência finita de instruções ou passos bem definidos, projetados para resolver um problema ou realizar uma tarefa. Ele pode ser implementado em diversas linguagens de programação e consiste em um conjunto de operações que, quando executadas em ordem, produzem um resultado.

Grupos de Estruturas Utilizadas em Algoritmos

Para implementar um algoritmo, utilizamos diversas estruturas de controle que organizam o fluxo de execução. As principais estruturas são:

  • Estrutura Sequencial: As instruções são executadas uma após a outra, na ordem em que aparecem.
  • Estrutura Condicional: Permite que o algoritmo tome decisões com base em condições. Exemplos: if, else e switch.
  • Estrutura Iterativa (ou de Repetição): Permite repetir um conjunto de instruções enquanto uma condição for verdadeira. Exemplos: for e while.
  • Estrutura Recursiva: Uma função que se chama a si mesma para resolver um problema, geralmente dividindo-o em subproblemas menores.

Detalhamento das Estruturas

Estrutura Sequencial

Na estrutura sequencial, cada instrução é executada em ordem, uma após a outra. Por exemplo, para fazer um sanduíche, você seguiria passos como: pegar os ingredientes, montar o sanduíche e servir.

Estrutura Condicional

Estruturas condicionais permitem que o algoritmo escolha um caminho diferente de execução baseado em uma condição. Por exemplo, em um algoritmo que decide se uma pessoa pode votar, você usaria uma estrutura if para verificar a idade.

Estrutura Iterativa (Repetição)

A iteração permite repetir um conjunto de instruções várias vezes até que uma condição seja atendida. Essa estrutura é fundamental para percorrer listas ou executar tarefas repetitivas.

  • Laço for: Utilizado para iterar sobre sequências (como listas ou strings) ou sobre um intervalo numérico gerado por range().
  • Laço while: Executa um bloco de código enquanto uma condição for verdadeira.

Exemplo de iteração com for:

lista = [1, 2, 3, 4, 5]
for i in range(len(lista)):
    print(lista[i])

Estrutura Recursiva

Na recursão, uma função chama a si mesma para resolver um problema, geralmente quebrando-o em partes menores. É essencial definir uma condição de parada para evitar loops infinitos. Um exemplo clássico é o cálculo do fatorial de um número.
