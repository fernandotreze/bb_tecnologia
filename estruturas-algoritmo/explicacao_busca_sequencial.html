Explicação Passo a Passo: Busca Sequencial

Definição da Função:

def busca_sequencial(lista, valor):
    def: Palavra-chave que indica a definição de uma função.
    busca_sequencial: Nome da função que descreve o que ela faz.
    lista: Parâmetro que representa a lista onde vamos procurar o valor.
    valor: Parâmetro que representa o valor que queremos encontrar na lista.

Loop (Laço de Repetição):

for i in range(len(lista)):
    for: Palavra-chave usada para iniciar um laço de repetição.
    i: Variável que servirá como índice para acessar cada elemento da lista.
    range(len(lista)): A função len(lista) retorna o tamanho da lista (quantidade de elementos).
                      O range() gera uma sequência de números de 0 até o tamanho da lista - 1.
                      Ou seja, o laço vai percorrer todos os índices da lista.

Verificação da Condição:

if lista[i] == valor:
    if: Condicional que verifica se a condição é verdadeira.
    lista[i]: Acessa o elemento da lista no índice i.
    valor: Compara o valor encontrado na lista com o valor que estamos procurando.
           Se a condição for verdadeira (ou seja, se o valor foi encontrado), o código dentro do if será executado.

Retorno do Índice:

return i
    Se o valor foi encontrado, a função retorna o índice i, onde o valor foi encontrado na lista.
    O código executa a função e para imediatamente ao encontrar o primeiro valor correspondente.

Valor Não Encontrado:

return -1
    Se o loop terminar e o valor não for encontrado (ou seja, se o if nunca for verdadeiro), a função retorna -1,
    indicando que o valor não foi encontrado na lista.

Resumo do Fluxo:

A função começa a percorrer a lista a partir do primeiro elemento (índice 0).

Para cada elemento, verifica se ele é igual ao valor que procuramos.

Se encontrar o valor, retorna o índice do elemento.

Se não encontrar o valor após percorrer toda a lista, retorna -1.

Esse algoritmo tem complexidade O(n), onde n é o número de elementos na lista, porque no pior caso ele vai percorrer todos os elementos para achar o valor.
