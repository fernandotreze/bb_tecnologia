Anotações de Leitura: Pascal e Python

Estas anotações resumem os principais pontos discutidos sobre a linguagem Pascal, estruturas de dados e a implementação de um algoritmo para inversão de uma sequência, com comparações e exemplos em Python.

1. Pascal: Contexto e Notações

Blaise Pascal foi um matemático, físico, filósofo e inventor francês, e sua influência se estende à área da computação. A linguagem Pascal, criada por Niklaus Wirth, destaca-se por sua clareza e estrutura organizada, sendo bastante utilizada para o ensino de algoritmos e programação estruturada.

1.1 Operador de Atribuição :=

Em Pascal, o símbolo := é utilizado para atribuir um valor a uma variável. Exemplo: 

var
  x: integer;
begin
  x := 10;  (* Atribui 10 a x *)
end.

2. Estruturas de Dados

São formas de armazenar e organizar dados na memória:

  • Variáveis Simples: Armazenam um único valor (ex.: idade: integer).
  • Vetores (Arrays Unidimensionais): Conjunto de elementos do mesmo tipo, acessados por índice (ex.: notas: array[1..5] of real).
  • Matrizes (Arrays Bidimensionais): Estruturas com duas dimensões (linhas e colunas), como uma tabela.
  • Registros (Records): Agrupam diferentes tipos de dados em uma única estrutura, útil para representar entidades complexas (ex.: dados de um aluno com nome, idade e nota).

3. Arrays Multidimensionais

É possível criar arrays com três ou mais dimensões (3D, 4D, 5D, etc.). Por exemplo, um array 3D em Pascal: 

var
  cubo: array[1..3, 1..3, 1..3] of integer;
begin
  cubo[1,2,3] := 42;
end.

Conforme aumentam as dimensões, a manipulação e visualização dos dados tornam-se mais complexas.

4. Ponteiros e Registros

Ponteiros são variáveis que armazenam endereços de memória. Em Pascal, podemos referenciar registros (estruturas com campos diversos) por meio de ponteiros utilizando o símbolo ↑ (ou ^ em algumas versões).

Exemplo: pt ↑.info representa o campo info de um registro localizado no endereço apontado por pt.

Além disso, a passagem de parâmetros por referência (usando var) permite que uma rotina modifique diretamente a variável original. Em algumas variações de Pascal, o símbolo % é usado para marcar comentários.

5. Algoritmo de Inversão de uma Sequência

O objetivo é inverter a ordem dos elementos de um vetor. A ideia é trocar o primeiro elemento com o último, o segundo com o penúltimo, e assim por diante, percorrendo apenas metade do vetor. 

Algoritmo 1.1
Inversão de uma sequência
para i := 1, …, ⎣n/2⎦ faça
    temp := S[i]
    S[i] := S[n – i + 1]
    S[n – i + 1] := temp

A notação ⎣x⎦ (piso de x) indica o maior inteiro menor ou igual a x. Assim, para um vetor com n elementos, se n for ímpar, o elemento central não é alterado.

Exemplo: Para S = [10, 20, 30, 40, 50]:

  • Troca do 1º com o 5º elemento: [50, 20, 30, 40, 10]
  • Troca do 2º com o 4º elemento: [50, 40, 30, 20, 10]
  • O elemento central (30) permanece inalterado.

6. Implementação em Python

Em Python, podemos inverter uma lista de duas formas:

6.1 Com Loop Manual

def inverter_lista(lista):
    n = len(lista)
    for i in range(n // 2):
        # Troca os elementos sem variável temporária usando atribuição múltipla
        lista[i], lista[n - i - 1] = lista[n - i - 1], lista[i]

# Exemplo de uso
lista = [10, 20, 30, 40, 50]
inverter_lista(lista)
print(lista)  # Saída: [50, 40, 30, 20, 10]

6.2 Usando Slicing

lista = [10, 20, 30, 40, 50]
lista = lista[::-1]
print(lista)  # Saída: [50, 40, 30, 20, 10]

O slicing cria uma nova lista invertida e percorre todos os elementos, enquanto o loop manual modifica a lista original e percorre apenas metade dos elementos.

7. Atribuição Múltipla e Uso das Vírgulas em Python

A linha a seguir pode parecer confusa à primeira vista: 

lista[i], lista[n - i - 1] = lista[n - i - 1], lista[i]

Ela utiliza atribuição múltipla para trocar os valores de duas posições da lista sem precisar de uma variável temporária:

  • lista[i] e lista[n - i - 1] (lado esquerdo) são as posições onde os valores serão armazenados.
  • lista[n - i - 1], lista[i] (lado direito) são os valores avaliados e trocados simultaneamente.

Essa sintaxe permite que Python realize a troca de valores de forma direta e limpa.

8. Diferença de Indexação: Pascal vs. Python

A diferença chave está na indexação:

  • Pascal: Vetores começam no índice 1. Assim, o primeiro elemento é S[1] e o último é S[n]. A troca usa S[n - i + 1] para acessar o elemento espelhado.
  • Python: Listas começam no índice 0. Portanto, o primeiro elemento é lista[0] e o último é lista[n - 1]. A troca usa lista[n - i - 1] para acessar o elemento espelhado.

Esse ajuste (+1 em Pascal e -1 em Python) garante que os algoritmos acessem corretamente os elementos correspondentes em cada linguagem.
