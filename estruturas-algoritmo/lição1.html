Estruturas de Dados e Algoritmos - Lição 1

1. O que são Estruturas de Dados?

As estruturas de dados são formas de organizar e armazenar dados de maneira eficiente. O tipo de estrutura que você escolhe depende do tipo de operação que você precisa realizar. Algumas das estruturas mais comuns incluem:

  • Arrays (ou Vetores): Armazenam dados em sequência. O acesso aos elementos é rápido, mas adicionar ou remover elementos no meio pode ser lento.
  • Listas Ligadas: Cada elemento aponta para o próximo. Inserir e remover elementos é fácil, mas o acesso aos dados é mais lento.
  • Pilhas (Stacks): Os elementos são inseridos e removidos do topo (LIFO - Last In, First Out).
  • Filas (Queues): Os elementos são inseridos no final e removidos do início (FIFO - First In, First Out).

2. O que são Algoritmos?

Algoritmos são sequências de instruções que resolvem um problema ou realizam uma tarefa. Quando falamos de algoritmos de busca e ordenação, estamos tratando de como buscar um elemento ou organizar dados de forma eficiente.

Vamos começar com dois tipos de algoritmos básicos:

  • Busca Sequencial: A busca sequencial percorre todos os elementos de uma lista até encontrar o item desejado.
  • Busca Binária: A busca binária é mais eficiente, mas funciona apenas em listas ordenadas. Ela divide a lista ao meio a cada comparação.

3. Exemplo de Busca Sequencial

A busca sequencial é o algoritmo mais simples. Ele verifica cada item da lista até encontrar o valor desejado.

 
                def busca_sequencial(lista, valor):
                for i in range(len(lista)):
                    if lista[i] == valor:
                        return i  # Retorna o índice do valor
                return -1  # Retorna -1 se o valor não for encontrado
            
            # Testando a função
            lista = [10, 20, 30, 40, 50]
            valor = 30
            resultado = busca_sequencial(lista, valor)
            print(f'O valor {valor} foi encontrado no índice {resultado}')

Veja a explicação detalhada

Este código percorre a lista até encontrar o valor desejado e retorna o índice do item. Se não encontrar o item, retorna -1.

4. Exemplo de Busca Binária

A busca binária é mais eficiente, mas só funciona em listas ordenadas. Ela divide a lista ao meio e, a cada comparação, elimina metade da lista.


def busca_binaria(lista, valor):
    esquerda, direita = 0, len(lista) - 1
    while esquerda <= direita:
        meio = (esquerda + direita) // 2
        if lista[meio] == valor:
            return meio  # Retorna o índice do valor
        elif lista[meio] < valor:
            esquerda = meio + 1
        else:
            direita = meio - 1
    return -1  # Retorna -1 se o valor não for encontrado

Veja a explicação detalhada

Esse algoritmo busca por um valor, dividindo a lista em duas metades até encontrar o número ou concluir que ele não está na lista.
